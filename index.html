Faça o download do JSON da página

Abaixo você encontrará um botão para fazer o download do JSON da página. O link te levará ao drive, basta apertar com o botão direito e selecionar a opção "Fazer download".

Crie uma nova página usando o Elementor

A página deve ser criada com as opções:

  • Nome da página: opcional (sugestão: Proposta comercial)
  • Modelo: Elementor Canvas
  • URL: opcional (sugestão: proposta-comercial)

Importe o modelo para sua página

Clique na seta para baixo ao lado do botão "Publicar", clique o ícone "Importar um modelo" e selecione o arquivo JSON que você baixou nos passos anteriores.

Estilizando a página

A página da proposta foi toda configurada e estilizada de maneira fácil e intuitiva, utilizando a menor quantidade possivel de widgets. Todos os estilos podem ser alterados de acordo com sua identidade visual.

Mudando o telefone do link da CTA

No penúltimo widget HTML da página, você vai encontrar um script onde é configurado o link da CTA. Altere a seguinte linha para o valor do whatsapp onde o cliente vai ser redirecionado para aceitar a proposta: 

                  const telefone = "Altere aqui (mantendo as aspas)";

Preencha os dados do formulário

Utilize o formulário disponibilizado nessa página para preencher os dados e gerar a proposta. Todos os campos tem nomes e placeholders legíveis e precisam ser preenchidos corretamente.

Campos iniciais

Nome do campo Descrição
Nome do cliente Define o nome do cliente que aparecerá na proposta
URL do seu dominio É a URL do wordpress onde está hospedado sua página da proposta
Slug da sua página O que aparece no link da proposta após a / . Ex.:www.seudominio.com/proposta-comercial
Data de envio A data que a proposta esta sendo enviada.
Data de validade A data que a proposta ficará disponível para seu cliente

Parametros comuns

A página já está configurada para receber todos os parametros comuns vindos do formulário, e caso queira mostrar uma informação novamente em outro lugar da página, deve colocar um <span data-param="value"></span> dentro do widget de texto/título. Alterando o valor dentro das aspas.
Siga a lista de abaixo para os valores padrões para cada parâmetro.

Valor do parâmetro Descrição
nome-cliente Nome do cliente que vai aparecer na proposta
data-envio A data de envio da proposta
data-validade A data de validade da proposta
servico Nome do serviço principal preenchido no campo "Serviço que será feito"
preco-total O valor do serviço - desconto aplicado
inclui Renderiza todos os items adicionados no campo "O que é incluido", deve ser usado da seguinte maneira: <div id="includeItemsWrapper" data-param="inclui"></div>

Criando novos parâmetros

No formulário há um botão de acrescentar parâmetros. Clique nele para abrir um campo para digitar o nome do parametro, digite o nome, clique em adicionar, e depois preencha o valor que esse novo parâmetro vai receber.

Usando o parametro no Elementor

Para usar o novo parametro em algum campo do elementor, você deve criar um widget de texto e colocar o valor do parametro em forma de slug no <span data-param="value"></span>
